### Краткое описание

Как пользоваться программой:
При запуске программы (команда "python B1-Cat-Mouse.py") в консоль нужно ввести индекс файла, который вы хотите открыть
К примеру, если вы хотите открыть файл с данными 3.ChaseData.txt, введите в консоль "3"

Также вы можете ввести свой файл в программу
Для этого вам нужно ввести -1
Затем следующей строкой введите название вашего файла
К примеру, если вы хотите открыть файл с данными MyChaseData.txt, введите в консоль "-1"
Затем в следующей строчке введите "MyChaseData.txt"
