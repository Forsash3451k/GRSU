### Краткое описание

Как пользоваться программой:
При запуске программы (команда "python B1-Emission-Monitoring.py") в консоль нужно ввести индекс файла, который вы хотите открыть
К примеру, если вы хотите открыть файл с данными Precip2.txt, введите в консоль "2"

Также вы можете ввести свой файл в программу
Для этого вам нужно ввести -1
Затем следующей строкой введите название вашего файла
К примеру, если вы хотите открыть файл с данными MyPrecipData.txt, введите в консоль "-1"
Затем в следующей строчке введите "MyPrecipData.txt"
